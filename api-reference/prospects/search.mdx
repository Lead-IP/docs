---
title: "Search Prospects"
api: "POST /v1/prospects/search"
description: "Search for prospects using advanced filters"
---

## Request

### Headers

```
Authorization: Bearer wbk_your_api_key_here
Content-Type: application/json
```

### Body Parameters

<ParamField body="query" type="object" required>
  Search query object with filters
  <Expandable title="Query fields">
    <ParamField body="person_titles" type="array">
      Job titles to search for (e.g., `["CEO", "CTO", "Founder"]`)
    </ParamField>
    <ParamField body="person_locations" type="array">
      Locations (e.g., `["United States", "United Kingdom"]`)
    </ParamField>
    <ParamField body="organization_industries" type="array">
      Industry categories (e.g., `["Legal", "Technology"]`)
    </ParamField>
    <ParamField body="organization_num_employees_ranges" type="array">
      Company size ranges: `["1-10", "11-50", "51-200", "201-500", "501-1000", "1001+"]`
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="page" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField body="per_page" type="integer" default="25">
  Results per page (max: 100)
</ParamField>

## Response

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="prospects" type="array">
      Array of prospect objects
      <Expandable title="Prospect object">
        <ResponseField name="id" type="string">
          Prospect ID from data provider
        </ResponseField>
        <ResponseField name="name" type="string">
          Full name
        </ResponseField>
        <ResponseField name="title" type="string">
          Job title
        </ResponseField>
        <ResponseField name="email" type="string">
          Email address (may be null)
        </ResponseField>
        <ResponseField name="company" type="string">
          Company name
        </ResponseField>
        <ResponseField name="location" type="string">
          Location (city, state/country)
        </ResponseField>
        <ResponseField name="linkedin_url" type="string">
          LinkedIn profile URL
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  Pagination metadata
  <Expandable title="properties">
    <ResponseField name="page" type="integer">
      Current page
    </ResponseField>
    <ResponseField name="per_page" type="integer">
      Results per page
    </ResponseField>
    <ResponseField name="total" type="integer">
      Total matching prospects
    </ResponseField>
  </Expandable>
</ResponseField>

```python Python
import requests

API_KEY = "wbk_your_api_key_here"
BASE_URL = "https://nbkxaqxwvkgbddekwsma.supabase.co/functions/v1/api-gateway"

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}

data = {
    "query": {
        "person_titles": ["CEO", "Chief Executive Officer"],
        "organization_industries": ["Legal"],
        "person_locations": ["United States"],
        "organization_num_employees_ranges": ["11-50", "51-200"]
    },
    "per_page": 50
}

response = requests.post(f"{BASE_URL}/v1/prospects/search", headers=headers, json=data)
prospects = response.json()["data"]["prospects"]
print(f"Found {len(prospects)} prospects")
```

## Query Examples

### Search by Multiple Job Titles

```json
{
  "query": {
    "person_titles": ["CTO", "VP of Engineering", "Director of Technology"]
  }
}
```

### Search by Location and Industry

```json
{
  "query": {
    "person_locations": ["California", "Texas"],
    "organization_industries": ["Technology", "SaaS"]
  }
}
```

### Search by Company Size

```json
{
  "query": {
    "organization_num_employees_ranges": ["51-200", "201-500"]
  }
}
```

## Notes

<Note>
  Prospect search results come from a third-party data provider. Email addresses may not always be available.
</Note>

<Tip>
  After finding prospects, use the [Save Prospects](/api-reference/prospects/save) endpoint to add them to your CRM.
</Tip>
